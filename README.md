# super-fortnight Project
## Description
The super-fortnight project is a Django-based web application for Mama Mary Lwate's Good Hope Community Organisation. The organisation was founded in 1997 to care for abused and abandoned children.
## Key Features
- User-friendly interface
- Awards and services sections
- Contact page
## Tech Stack
- Django
- Python
- SQLite database
- Bootstrap
- HTML/CSS
## Installation
1. Clone the repository: git clone https://github.com/username/super-fortnight.git
2. Install dependencies: pip install -r requirements.txt
3. Run migrations: python manage.py migrate
4. Run the application: python manage.py runserver
## Usage
1. Open a web browser and navigate to http://localhost:8000
2. Explore the website and its features
## Environment Variables
- SECRET_KEY: a secret key for the Django application
- DEBUG: a boolean indicating whether the application is in debug mode
